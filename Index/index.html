Brad's Website

Hi, my name is Brad Frye, welcome to my website. This site is run on a LAMP stack hosted on Amazon Web Services (AWS). I'm using this site to share programs I've written with friends and family, and to develop a portfolio to demonstrate my experience in HTML, CSS, JavaScript, PHP, and MySQL.

Currently, you may view my resume, play UNO, or try out the Shopping List or Thesaurify programs (click the orange Instructions box on any Program page if you need help using it.) You may also leave me a comment at the bottom of this page if you'd like - let me know if you have any questions, suggestions, or problems with the site. Please keep it clean!

I have more content in progress, and will add it to the site when I have time.
